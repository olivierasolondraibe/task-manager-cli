## 🎯 Vue d'ensemble

Task Manager CLI est une application en ligne de commande permettant de gérer vos tâches de manière simple et efficace. Elle offre une interface intuitive pour créer, modifier, supprimer et organiser vos tâches.

## 📦 Installation

### Prérequis

- Dart 3.12.2 ou supérieur

### Démarrage rapide

```bash
# Clonez le repository
git clone https://github.com/olivierasolondraibe/task-manager-cli.git
cd task-manager-cli

# Installez les dépendances
dart pub get

# Exécutez l'application
dart run

#Exécutez le test
dart test
